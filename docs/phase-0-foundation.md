# Phase 0 foundation

Первый инкремент покрывает только стартовую платформенную основу.

## Что входит

- структура репозитория
- контейнерная инфраструктура
- web-shell для role-based dashboard
- API-заготовка под `legal requests`
- контракты и словарь первых ролей

## Что не входит

- persistent storage
- полноценный workflow engine
- AI-функции
- интеграции с внешними системами
- production-ready security

## Первый demo-сценарий

1. Бизнес-заказчик открывает портал.
2. Видит entry-point для подачи юридической заявки.
3. Отправляет заявку с типом, приоритетом и описанием.
4. API возвращает карточку заявки и статус `new`.
5. Юрист видит, какие заявки должны появляться в его очереди на следующем инкременте.
